# Runtime API 示例

本页面演示了 VitePress 提供的一些运行时 API 的使用方法。

主要的 `useData()` API 可用于访问当前页面的站点、主题和页面数据。它可以在 `.md` 和 `.vue` 文件中工作：

```html
<script setup>
import { useData } from 'vitepress'
import { useSidebar } from 'vitepress/theme'

const { theme, page, frontmatter } = useData()
const {hasSidebar} = useSidebar()
</script>

## 结果示例

### 主题数据
<pre>{{ theme }}</pre>

### 页面数据
<pre>{{ page }}</pre>

### 页面 Frontmatter 数据
<pre>{{ frontmatter }}</pre>

### 是否有侧边栏
<pre>{{ hasSidebar }}</pre>
<template>
  <div>
    <h1>当前页面是否有侧边栏</h1>
    <p>是否有侧边栏： {{ hasSidebar }}</p>
    <p>侧边栏内容： {{ sidebar }}</p>
    <p v-if="hasSidebar">当有侧边栏内容时： {{ sidebar }}</p>
  </div>
</template>
```

<script setup>
import { useData } from 'vitepress'
import { useSidebar } from 'vitepress/theme'

const { theme, page, frontmatter } = useData()
